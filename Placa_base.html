Placa Base

Una placa base, también conocida como motherboard o tarjeta madre, es el componente principal de un sistema informático. Actúa como el esqueleto que conecta y permite la comunicación entre todos los demás componentes de hardware, como la CPU, la memoria RAM, las tarjetas gráficas, y los dispositivos de almacenamiento. La placa base está compuesta por una serie de circuitos impresos que facilitan la transmisión de datos y la energía entre los distintos elementos.

Las placas base vienen en varios factores de forma, siendo ATX, microATX y mini-ITX los más comunes. Cada factor de forma se adapta a diferentes necesidades de espacio y rendimiento, permitiendo una mayor o menor cantidad de ranuras de expansión para agregar dispositivos adicionales, como tarjetas gráficas o de sonido.

Uno de los componentes clave de la placa base es el chipset, que gestiona la comunicación entre la CPU, la memoria y los periféricos. También alberga la BIOS o UEFI, el software que inicializa el hardware durante el arranque del sistema.

La placa base incluye puertos y conectores para dispositivos externos como USB, HDMI y Ethernet. Su elección es fundamental para garantizar la compatibilidad y el rendimiento óptimo de todo el sistema, ya que determina qué componentes pueden instalarse y cómo interactúan entre sí.

Componentes de la Placa Base

  • Socket de CPU
  • Chipset
  • Ranuras de RAM
  • Ranuras de expansión (PCIe)
  • Conectores de almacenamiento (SATA, M.2)
  • Conectores de alimentación
  • Puertos USB
  • Conector ATX de 24 pines
  • Conectores de audio
  • Puertos de red (Ethernet)
  • BIOS/UEFI
  • Batería CMOS
